# ScooterKey PWA — Deploy to GitHub Pages (Free Hosting)
# ─────────────────────────────────────────────────────────────────────────────
# Result: your app lives at https://YOUR_USERNAME.github.io/scooterkey/
# Open that URL in Chrome on Android → tap "Add to Home Screen" → works like an app
# ─────────────────────────────────────────────────────────────────────────────

STEP 1 — Create a GitHub account (if you don't have one)
  → https://github.com/signup

STEP 2 — Create a new repository
  → https://github.com/new
  Name it: scooterkey
  Set to: Public
  Click: Create repository

STEP 3 — Upload the 3 files
  In the new repo, click "Add file" → "Upload files"
  Upload: index.html, manifest.json, sw.js
  Click: Commit changes

STEP 4 — Enable GitHub Pages
  Go to repo Settings → Pages
  Source: Deploy from a branch
  Branch: main  /  folder: / (root)
  Click: Save

STEP 5 — Open your app (takes ~2 min to go live)
  URL: https://YOUR_USERNAME.github.io/scooterkey/

STEP 6 — Install on phone
  Open URL in Chrome on Android
  Tap the 3-dot menu → "Add to Home screen"
  → It installs like a real app, works offline

IMPORTANT — Chrome Bluetooth requirement:
  Must use Chrome browser on Android (not Samsung Internet, Firefox, or Safari)
  The BLE scan button triggers the native Android Bluetooth device picker
